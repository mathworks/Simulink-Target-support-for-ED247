Getting Started with ED247 for Simulink

Description

Simulink Library to use ED247 protocol in models.

System Requirements

Download the ED247 library sources from GitHub and compile the library following instructions.
Install a supported compiler (For Windows : MinGW64 4.9.2, POSIX version)

Features

Additional Simulink library : Airbus ED247

Examples

Demos are available for different communication protocols:
  • A429
  • A664
  • A825
  • ANA (Analogic)
  • DIS (Discrete)
  • NAD (Non-Avionic Data)
