Package a toolbox

Describe the steps to follow for a developer to package and share ED247 for Simulink as a custom toolbox.

Prerequisites

Open the project by double-clicking on ED247_for_simulink.prj
Recommended
testRun('Mode','dev')

Package the Configuration App

Open (double-click) the project ED247 Configuration.prj in app/ folder.
Update the application version as needed.
Click on Package.

Update the metadata

edit .metadata
Update version and date
edit CHANGELOG.md
Update the CHANGELOG.

Package the toolbox

Open the toolbox packaging toolbox by double-clicking on ED247_for_simulink_tlbx.prj.
Update toolbox version
Copy-paste CHANGELOG.md content in Description field.

Share the toolbox

Share the file ED247 for Simulink.mltbx
