Installation guide

Prerequisites

Download the ED247 library sources from GitHub and compile the library following instructions.
Install a supported compiler (For Windows : MinGW64 4.9.2, POSIX version)

Installation

Double-click on toolbox installer file (ED247_for_simulink.mltbx).

Configuration

To configure the MinGW64 compiler installation folder, launch the configuration UI
Via App tab
Via command line
ed247.ConfigurationUI.open()
Notes : MinGW folder should contain bin subfolder

Initialization

ED247 for Simulink is automatically configured at Simulink startup.
